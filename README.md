## วัตถุประสงค์ของรายวิชา
- ศึกษาแนวคิดและหลักการพัฒนา Mobile Web Application
- ฝึกใช้งาน Framework และ Library ที่นิยมในอุตสาหกรรม
- พัฒนาแอปพลิเคชันที่สามารถใช้งานบน Mobile Device ได้จริง
- เรียนรู้การเชื่อมต่อ Backend และ Cloud Service
